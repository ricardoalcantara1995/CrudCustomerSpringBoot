# CrudCustomerSpringBoot

Este es un proyecto de Spring Boot como BackEnd y Angularcomo FrontEnd que implementa un CRUD completo para la entidad `customer`. 

Utiliza MySQL como base de datos.

## Tecnologías
- Java (Spring Boot)
- Angular 15
- MySQL
- Maven
